# Trust Level Avatar Flair - Mecha 3.1 / v6 Icons

This build combines the v3.1 mecha SVG badge shell with the v6 internal symbols.

- Outer badge and metal facets: v3.1
- Internal TL0-TL4 symbols: v6
- Topic avatar size: 17 x 19 px
- User card/profile size: 34 x 38 px

Install as a Discourse theme component. Keep `Badge Style` set to `metal-shield` and `Use Font Awesome` disabled for the built-in SVG mode.
